
# Chrome 扩展

[[toc]]

## 扩展说明

扩展的作用是从数据源获取所需要的简单数据，并转化为对应存储的数据结构。

## 操作步骤

```mermaid
打开豆瓣读书 --> 右键 --> 保存图书信息
```

## 数据源

* [豆瓣读书](https://book.douban.com)
* [QQ音乐](https://y.qq.com)
